   Lista não rola

✅ AGORA:
   Trackpad com dois dedos → Scroll suave ✓
   Roda do mouse → Funciona ✓
   
🔧 SOLUÇÃO:
   Bug identificado: CustomTkinter 5.2.2 tem problema com scroll no macOS
   
   Solução aplicada:
   1. Detecta automaticamente o sistema operacional (Mac/Windows/Linux)
   2. Vincula eventos de scroll diretamente ao canvas interno
   3. Ativa/desativa scroll quando mouse entra/sai da área
   
   Código implementado:
   - Método _setup_scroll() que configura eventos multiplataforma
   - Bind de <MouseWheel> no macOS
   - Bind de <Button-4> e <Button-5> no Linux
   - Handler específico para cada SO

📝 Arquivo modificado: src/ui.py

⚠️  IMPORTANTE: 
   Mova o mouse SOBRE a lista antes de rolar!
   O scroll só funciona quando o cursor está na área da lista.
